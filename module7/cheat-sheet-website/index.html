My Coding Cheatsheet

Html Tags

Name Tag Description
Head <head> </head> The HTML <head> element contains machine-readable information (metadata) about the document, like its title, scripts, and style sheets.
Link <link> The HTML External Resource Link element (<link>) specifies relationships between the current document and an external resource. This element is most commonly used to link to stylesheets, but is also used to establish site icons (both "favicon" style icons and icons for the home screen and apps on mobile devices) among other things.
Meta <meta> The HTML <meta> element represents metadata that cannot be represented by other HTML meta-related elements, like <base>, <link>, <script>, <style> or <title>
Style <style><style/> The HTML <style> element contains style information for a document, or part of a document.
Title <title><title/>/td> The HTML Title element (<title>) defines the document's title that is shown in a browser's title bar or a page's tab.

Html Attributes

Name Format Description
Class class="" Assigns a class to an element to be targeted by CSS. Mulitple elements can have the same class.
ID id="" Assigns a spesfic id to the tag from CSS to target with the highest priority. Can only be assigned to a single tag.
HyperText Reference href="" Establishes the target of a hyperlink.
